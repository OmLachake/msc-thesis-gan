# GANs for Panoramic Image Extrapolation

The following code is a modified version of : [https://github.com/basilevh/image-outpainting](https://github.com/basilevh/image-outpainting)

It uses Hue Loss, Perceptual Loss and different masks for analyzing the effects of loss functions on shallow and linear generative networks.

This is not meant to be used for any commercial purposes.
